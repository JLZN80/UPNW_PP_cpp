# UPNW_PP_cpp

Repositorio de ejercicios, ejemplos y desarrollos en C++ para el curso de
**Principios de Programacion** de la **Universidad Privada Norbert Wiener
(UPNW)**.

Este espacio reune codigos trabajados en clase, practicas semanales, ejercicios
tipo parcial y pequenos retos para que los estudiantes puedan practicar,
equivocarse, corregir y fortalecer su razonamiento algoritmico.

## Proposito

El objetivo de este repositorio es dejar a disposicion de los estudiantes una
base de ejercicios progresivos que les ayude a:

- Practicar la sintaxis basica de C++.
- Comprender variables, operadores, condicionales, ciclos, funciones y archivos.
- Resolver problemas mediante pensamiento logico y algoritmico.
- Revisar ejemplos similares a los trabajados en clase.
- Prepararse para practicas, controles y evaluaciones parciales.
- Comparar distintas formas de plantear una solucion.
- Modificar programas existentes para explorar nuevos casos.

La intencion no es solo entregar codigo terminado, sino ofrecer material que
sirva como punto de partida para analizar, ejecutar, depurar y mejorar
soluciones.

## Como usar estos ejercicios

Se recomienda que los estudiantes no copien directamente las soluciones. Una
mejor forma de trabajar es:

1. Leer el enunciado o identificar que problema resuelve el codigo.
2. Intentar resolverlo primero sin mirar la solucion completa.
3. Ejecutar el programa y probar distintos casos.
4. Modificar entradas, condiciones o formulas para observar cambios.
5. Reescribir la solucion con sus propias palabras y estilo.
6. Agregar validaciones o mejoras cuando sea posible.
7. Comparar su solucion con la version del repositorio y corregir diferencias.

## Como compilar

Para compilar un archivo de C++ con `g++`, se puede usar un comando como:

```bash
g++ "ExParcial/Ejercicio_42_CalculadoraGym.cpp" -o programa
```

Luego se ejecuta:

```bash
./programa
```

En Windows, si se genera un archivo `.exe`, puede ejecutarse asi:

```powershell
.\programa.exe
```

Tambien se puede trabajar desde editores como Visual Studio Code, Dev-C++,
Code::Blocks u otro entorno configurado para C++.

## Recomendaciones para estudiantes

- Ejecuta el codigo, no te quedes solo leyendolo.
- Cambia valores y prueba casos limite.
- Comenta las lineas que no entiendas y revisalas en clase.
- Practica escribir el mismo ejercicio desde cero.
- No memorices soluciones: entiende el proceso.
- Usa estos ejemplos como guia, no como reemplazo de tu propio razonamiento.
- Revisa errores de compilacion con calma: suelen indicar exactamente donde
  empezar a corregir.

## Buenas practicas sugeridas

- Usa nombres de variables claros y relacionados con el problema.
- Mantiene una indentacion consistente.
- Divide problemas grandes en funciones cuando sea conveniente.
- Valida entradas cuando el ejercicio lo requiera.
- Prueba el programa con datos normales, extremos y erroneos.
- Guarda versiones de tus avances para poder comparar cambios.
